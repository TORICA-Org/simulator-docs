# 概要
鳥人間コンテスト 滑空機部門における人力飛行機の飛行を再現した，パイロット練習用シミュレーターである．

# 意義
以下の課題を解決することにある．
- 鳥人間コンテスト 滑空機部に出場する自作人力飛行機はその多くが着陸機構を持たず，実際の機体の運動を体験することが困難なこと
- ピッチング制御にパイロットの体重移動を採用している場合，その練習が困難なこと
- 大会本番で起こるトラブルの対策が困難なこと

# 要求
- 風の向きや強さ，離陸角度などのパラメーターを任意に設定できる．
- パイロットの体重移動による重心位置の変化により，シミュレーター内の機体の挙動を変化させることができる．
- VRヘッドセットを用いて，パイロットの視点映像を高い没入感で体験できる．
